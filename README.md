# Oh-My-Life-Patchy

> 一个专门识别并修复日常隐性损耗的 AI Agent  
> An AI agent that identifies and repairs daily hidden frictions.

## 项目简介

Oh-My-Life-Patchy 不是一个追求“无所不能”的通用助手，而是一只专门处理生活中“小而高频损耗”的龙虾。

它关注的不是宏大目标，而是那些“不严重、但天天耗人”的问题，例如：

- 消息看到了却迟迟不敢回
- 事情知道该做，却迟迟起不了头
- 通知和规则看完了，还是不知道自己该做什么
- 协作时进度失真，不知道怎么和队友同步现状
- 临近截止，脑子很乱，不知道先救哪一步
- 一些生活中的小摩擦不断消耗注意力和精力

Oh-My-Life-Patchy 的目标不是替用户管理整个人生，而是用最小、即时、可执行的方式，把这些持续消耗人的“隐性损耗”补上。

---
## 成熟版形态设想：便携具身化生活补丁玩偶

Oh-My-Life-Patchy 的成熟版不仅是一个软件 Agent，也可以扩展为一个可随身携带的具身化生活补丁终端。

它支持：

- **听**：微型麦克风作为“耳朵”，在用户授权和可控开关前提下获取低侵入生活线索
- **看**：微型相机作为“眼睛”，感知简单环境与任务场景
- **说**：微型扬声器作为“嘴巴”，主动提醒、播报补丁建议
- **写**：迷你打印机作为“手”，把“今晚最低完成版”“出门前清单”“待发送消息草稿”等关键补丁打印成实体纸条
- **走**：智能玩具车作为“脚”，让龙虾从桌面助手变成可移动的现实伙伴

其核心控制中枢可由**微型电脑 + 便携电源 + 便携 Wi-Fi**组成，支持与手机、电脑通讯，并内置龙虾主 Agent，使其从“屏幕里的 AI”进一步发展为“现实中的生活补丁伙伴”。

## 核心定位

**Oh-My-Life-Patchy = 一个专门识别并修复日常隐性损耗的 AI Agent。**

它的核心能力是：

- 识别用户当前卡住的是哪类问题
- 路由到对应 skill
- 输出最小、即时、可执行的补丁方案

它不强调“懂很多”，而强调：

- 补得准
- 补得快
- 补得能立刻用

## 项目结构

本项目采用：

**1 个主 Agent + 8 个 Skills**

### 主 Agent
负责：
- 识别当前问题类型
- 调用对应 skill
- 在复杂场景中做多技能协同编排

### 8 个 Skills

#### 沟通补丁
- **回消息急救**：处理已读未回、延迟回复、补救回复
- **体面表达**：处理催促、拒绝、改期、补救、重新联系
- **退出机制**：处理想退出、接不住、去不了、做不到的场景

#### 行动补丁
- **起手代打**：处理启动困难、开头困难、迟迟开始不了
- **未来减麻烦**：帮助用户用低成本动作减少未来的混乱和翻车

#### 判断补丁
- **行动翻译器**：把通知、规则、说明翻译成“你现在该做什么”
- **后悔模拟器**：在行动前预演短期后悔点，给出更稳妥方案

#### 生活补丁
- **找东西推理**：帮助用户更高效地找回常用物品，减少无序乱找

## 当前重点实现

当前版本重点实现了 3 个核心高频 skill：

### 1. 回消息急救
针对：
- 消息拖着没回
- 越拖越不敢回
- 想补救回复
- 想礼貌拒绝或留余地表达

输出：
- 可直接发送的回复版本
- 更稳妥 / 更自然 / 更简洁的表达方案

### 2. 起手代打
针对：
- 知道要做，但就是开始不了
- 题目有了，但标题、结构、开头都起不出来
- 截止临近，只想先启动一个最小版本

输出：
- 可直接开写的题目
- 最小结构
- 第一段草稿
- 最低交付线

### 3. 行动翻译器
针对：
- 通知看完还是不知道自己该干嘛
- 规则太长、条件太多、身份判断复杂
- 临近截止但优先级混乱

输出：
- 是否与用户有关
- 现在必须做什么
- 最晚什么时候做完
- 最容易漏掉的坑
- 今天最小行动清单

## 主 Agent 的复合场景能力

Oh-My-Life-Patchy 不只是单点工具，也支持复杂情境下的多技能协同。

例如在以下复合场景中：

- 协作对象已经来问进度，但自己还没开始
- 通知没完全看懂，任务要求也不明确
- 临近截止，时间有限，只能先止损
- 同时需要处理沟通、信息理解、启动困难

主 Agent 会优先按如下顺序处理：

1. 识别问题结构
2. 先生成状态同步消息
3. 再把要求翻译成最低交付线
4. 再给时间受限下的止损计划
5. 最后提醒最容易翻车的点

## 项目价值

Oh-My-Life-Patchy 解决的不是“高难技术问题”，而是更贴近日常生活的真实损耗：

- 沟通拖延
- 启动困难
- 信息理解成本
- 协作失真
- 小型生活摩擦

它的价值在于：

- 把空泛建议变成可执行补丁
- 把“我知道该做什么”变成“我现在就能开始”
- 把持续消耗人的小麻烦转化为可被修复的具体动作

---

## 安全与边界

Oh-My-Life-Patchy 不用于：

- 欺骗、操控、骚扰、PUA
- 非法用途
- 医疗、法律、财务等高风险专业替代
- 伪造、侵权、恶意误导

在敏感场景中，默认采用更保守、更稳妥的策略。

## 未来发展方向

未来版本中，Oh-My-Life-Patchy 可进一步扩展为具身化生活补丁终端，例如：

- 在用户授权和可控开关前提下，探索更自然的陪伴式生活补丁体验
