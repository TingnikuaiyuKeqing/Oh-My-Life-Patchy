# Oh-My-Life-Patchy 使用指南

## 1. 这是什么

Oh-My-Life-Patchy 是一个专门识别并修复日常隐性损耗的 AI Agent。

它不追求无所不能，而是专门处理那些“不严重、但天天耗人”的生活小损耗，例如：
- 消息拖着不回
- 任务起不了头
- 通知看不懂
- 协作进度失真
- 临近截止不知道先补哪一步

---

## 2. 怎么使用

你可以用两种方式使用它：

### 方式一：直接说你的问题
直接输入你现在卡住的情况，例如：

- “导师两天前给我发消息，我一直没回，现在怎么补救？”
- “我明天要交课程汇报，但标题和开头都起不出来。”
- “这份通知我看完还是不知道我到底要交什么。”
- “我和队友协作的项目掉进度了，我现在不知道怎么同步情况和补救。”

主 Agent 会自动判断这是哪类隐性损耗，并调用对应 skill。

### 方式二：进入指定模式
如果你已经知道自己要用哪种能力，也可以直接进入指定模式，例如：

- 回消息急救
- 起手代打
- 行动翻译器
- 体面表达
- 退出机制
- 未来减麻烦
- 后悔模拟器
- 找东西推理

---

## 3. 当前重点可用能力

### 回消息急救
适合：
- 已读不回
- 延迟回复
- 想补救一条消息
- 想礼貌拒绝或留余地表达

输出：
- 可直接发送的消息版本
- 更自然 / 更稳妥 / 更简洁版本

### 起手代打
适合：
- 事情知道要做，但就是开始不了
- 标题、结构、第一段都卡住
- 临近截止，只想先起一个最低可交版本

输出：
- 可直接开写的题目
- 最小结构
- 第一段草稿
- 最低交付路径

### 行动翻译器
适合：
- 通知、规则、说明看完还是不知道该干嘛
- 条件很多，身份判断复杂
- 截止时间和格式细节容易漏

输出：
- 这和你有没有关系
- 现在最该做什么
- 最晚什么时候做完
- 最容易漏的点
- 今天最小行动版清单

---

## 4. 最推荐的输入方式

为了让结果更准，建议尽量带上这几类信息：

- 场景是什么
- 对方是谁（如果涉及沟通）
- 你现在卡在哪
- 截止时间
- 你最担心什么
- 你想要的结果

例如：
“我和一个不太熟的人合作比赛，我之前答应做实验但忘了，现在想先和他说清楚，再给我一个两天最小推进计划。”

---

## 5. 风格适配

如果你愿意，可以额外提供：
- 你平时常发的两三句话
- 你喜欢更正式 / 更简洁 / 更委婉 / 更像聊天的风格

这样输出会更像你平时真的会说的话。

---

## 6. 使用原则

Oh-My-Life-Patchy 的默认工作方式是：

1. 先识别你卡住的是哪类隐性损耗
2. 先给你一个能立刻用的版本
3. 再给你最小补救路径
4. 最后才考虑优化、风格适配和展开

它优先做的是“补丁”，不是长篇分析。
